## Description
This is a Messenger App created using Firebase services. 
### How it works:
* The user makes login/register process ->  Clicks plus button to choose the user to message -> Starts messaging
* Two users can text each other

## Features
### Authentication
* Register using email, password, username, full name & profile picture.
* Login using email and password.
* Login & Logout
### Messaging
* Start conversations with any user that is in database
* Send text messages
* View personal page

* 100% Programmatic UI
* UIKit
* AutoLayout
* Firestore: Authentication, Cloud Firestore, Database
* JGProgressHUD
* MVVM pattern
* CocoaPods

## Setup
To run this project, install necessary pods using Terminal:

Set the cd to the project folder. 

Install pods by running this command in Terminal
```
pod install
```
